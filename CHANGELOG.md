# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-10-23

### Added
- Initial release of HowNow
- Core calculation tracking with BigNumber precision
- Builder pattern API with `.as()` method for contextual operations
- Internationalization support with token-based translations
- Debug mode with zero-overhead when disabled
- Full TypeScript support with type definitions
- Math operations: add, subtract, multiply, divide, floor, round, max
- Labeling methods: rollUp, label, step, branch
- Output methods: valueOf, toString, toDisplayFormat, getSteps
- Comprehensive README with examples
